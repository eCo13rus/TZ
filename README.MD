## Установка

- Php 8.0+

- Composer

- git clone https://github.com/eCo13rus/TZ.git

- cd TZ/

- composer install

## Запуск

- В файле test.txt пишем нужные слова 
- Далее в терминале пишем php index.php для разложения слов
- или пишем
- make start далее переходим на веб сервер для загрузки текстового файла через интерфейс.
